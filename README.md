# Weathercrawler

Automatic weather data extraction from http://wttr.in

## Installation

tested with:
  * Python 3.7.3
  * Python 3.9.4

````
git clone https://github.com/mrcgndr/weathercrawler.git .
cd weathercrawler
python3.9 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
````
